# How to use
Run the main.cpp
- void _pro:data process
- void _kmeans:k-means cluster
- void _cure:cure cluster
- void _DBSCAN:DBSCAN cluster
- void _kmeans++:k-means++ cluster

- data1_vec: data1_t's vector version

- data2_vec: data1_t's vector version
